# Ieso Nagata - Website

Este repositório contém o código-fonte e os arquivos de configuração do site oficial do **Ieso Nagata**. A página é hospedada diretamente através do GitHub, garantindo um ambiente estável e de fácil manutenção.

## 🌐 Link de Acesso
O site pode ser visualizado em tempo real através do link oficial:
👉 **[https://iesonagata.com.br](https://iesonagata.com.br/)**

## 📂 Estrutura do Repositório
O projeto está organizado na branch `master` e inclui:
- **Arquivos HTML/CSS/JS:** Estrutura, estilo e interatividade do site.
- **Ativos (Images/Assets):** Imagens e recursos visuais utilizados no layout.
- **Configurações de Hospedagem:** Arquivos necessários para o funcionamento no ambiente atual.

## 🛠️ Tecnologias Utilizadas
Este site foi desenvolvido com tecnologias web padrão para garantir performance e compatibilidade:
- HTML5
- CSS3
- JavaScript
